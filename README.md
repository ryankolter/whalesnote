
<div align="center">
  <h1>Whalenote (desktop version)</h1>
  <p>A cross-platform desktop note-taking software. Easily classify and store tens of thousands of notes.</p>
</div>

<br>

## Install

- in root directory

  ```bash
  npm i
  ```

- in app directory

  ```bash
  npm i
  ```

<br>

## development

- in root directory

  ```bash
  npm run start
  ```
  - the web server will run on port 3005

- in app directory
  ```bash
  npm run electron
  ```

<br>

## building && packaging

-  in root directory

  ```bash
  npm run build
  ```

-  in app directory

  ```bash
  // for windows
  npm run dist-win64
  // for mac
  npm run dist-mac
  ```
